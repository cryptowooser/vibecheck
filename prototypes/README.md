# Phase 1 Prototypes

Standalone browser-API prototypes for `docs/IMPLEMENTATION.md` WU-03 through WU-08.

Each prototype is independent and serves on `http://localhost:8080`.
Run from its directory:

```bash
./test.sh
```

Prototype map:
- `websocket-reconnect/` (WU-03)
- `media-recorder/` (WU-04)
- `push-notifications/` (WU-05)
- `camera-capture/` (WU-06)
- `pwa-install/` (WU-07)
- `speech-synthesis/` (WU-08)
